# baressg

A barebones static site generator that generates HTML from Markdown using lowdown. For more features, look into [Zola](https://github.com/getzola/zola), [Hugo](https://github.com/gohugoio/hugo), or [Jekyll](https://github.com/jekyll/jekyll).

[Live demo](https://alexh.dev/)

[How to use](https://github.com/aleksify/baressg/wiki/Getting-Started)
